🎯 三大核心创新函数

【第一大创新】findNextCluster
  复杂度: ⭐⭐⭐⭐⭐
  意义: Fuel的核心算法创新 - 多聚类动态选择
  输入参数:
    - cur_pos (Vector3d) - 当前位置
    - cur_vel (Vector3d) - 当前速度
    - cur_yaw (Vector3d) - 当前偏航角
    - neighbor (bool) - 是否考虑邻近聚类
  输出参数:
    - check_tour (vector<checkPoint>&) - 检查点路线
    - next_cluster_pos (Eigen::Vector3d&) - 下一聚类位置
  特点:
    ✓ 支持两种选择策略 (惯性规划 + TSP优化)
    ✓ 自动选择最低成本策略
    ✓ 实现多聚类分层规划

【第二大创新】updateInertialTour
  复杂度: ⭐⭐⭐⭐
  意义: Fuel独有的惯性路线规划
  输入参数:
    - indices (vector<int>) - 聚类索引
    - cluster_centers (vector<Eigen::Vector3d>) - 聚类中心
  输出参数:
    - inertia_cost (double&) - 惯性成本
  特点:
    ✓ 考虑机器人运动连续性
    ✓ 支持特征点距离阈值
    ✓ 可调节成本偏移参数

【第三大创新】solveTSP
  复杂度: ⭐⭐⭐⭐
  意义: 统一的TSP求解框架
  输入参数:
    - cost_matrix (Eigen::MatrixXd) - N×N成本矩阵
    - config (TSPConfig) - 求解配置
  输出参数:
    - result_indices (vector<int>&) - 最优路线
    - total_cost (double&) - 总成本
  特点:
    ✓ 灵活的配置系统
    ✓ 自动文件管理 (.tsp→求解→.txt)
    ✓ 支持多种问题类型
